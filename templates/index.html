Information

Click on grid cells to set:

  1. Start position (green)
  2. End position (red)
  3. Obstacles (gray) - Limited to n-2 obstacles

Generate policies using one of the following methods:

  • Random Policy - Completely random directions
  • Generate Path - Find a valid path with random start direction
  • Optimal Path - Find the best path with optimal start direction

Values are automatically calculated when a policy is generated.
